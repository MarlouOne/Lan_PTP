# для запуска теста передачи

1 - поменять в сервере bind(свой айпи)
2 - поменять в клиенте connect(айпи сервера)
3 - запустить сервер
4 - запустить клиент (посылать что-либо в данных файлах
	может только клиент)
5 - сообщение -f переводит клиент и сервер в режим передачи файлов
6 - ввести путь до файла с расширением
7 - произойдёт обмен данными
8 - будет получено сообщении об успешной доставке или нет(насччёт нет не помню)
9 - сообщение -q выключит и сервер и клиент(предположительно)
